🧩 Οδηγός Besu EduNet

Εγκατάσταση MetaMask · Σύνδεση Δικτύου · Mint NFT

🦊 1. Εγκατάσταση MetaMask

  1. Μεταβείτε στο metamask.io και πατήστε Download.
  2. Εγκαταστήστε την επέκταση στο Chrome / Firefox / Edge.
  3. Επιλέξτε Create a new wallet και αποθηκεύστε τη φράση ανάκτησης.
  4. Μην τη μοιραστείτε με κανέναν – είναι το “κλειδί” σας στο blockchain.

🧩 2. Προσθήκη Δικτύου Besu EduNet

  1. Ανοίξτε το MetaMask → Settings → Networks → Add network manually.
  2. Συμπληρώστε:
  • Network Name: Besu EduNet
  • RPC URL: https://rpc.dimikog.org/rpc/
  • Chain ID: 2025
  • Currency Symbol: EDU
  • Block Explorer: https://explorer.dimikog.org

Πατήστε Save και επιβεβαιώστε ότι το MetaMask εμφανίζει το δίκτυο “Besu EduNet”.

🪙 3. Mint του Course Starter NFT

  1. Μεταβείτε στη σελίδα mint: https://besu.dimikog.org/welcome-nft
  2. Πατήστε Connect Wallet → επιλέξτε MetaMask.
  3. Πατήστε Mint NFT και περιμένετε επιβεβαίωση συναλλαγής.
  4. Ελέγξτε το NFT σας στο Blockscout ή μέσα από το MetaMask.

Αυτό το NFT λειτουργεί ως ψηφιακή απόδειξη συμμετοχής στο μάθημα Web3 & Κρυπτογραφία.
