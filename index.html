🧩 Οδηγός Besu EduNet

Εγκατάσταση MetaMask · Σύνδεση Δικτύου · Mint NFT

🦊 1. Εγκατάσταση MetaMask

  1. Μεταβείτε στο metamask.io και πατήστε Download.
  2. Εγκαταστήστε την επέκταση στο Chrome / Firefox / Edge.
  3. Επιλέξτε Create a new wallet και αποθηκεύστε τη φράση ανάκτησης.
  4. Μην τη μοιραστείτε με κανέναν – είναι το “κλειδί” σας στο blockchain.

🧩 2. Προσθήκη Δικτύου Besu EduNet

  1. Ανοίξτε το MetaMask → Settings → Networks → Add network manually.
  2. Συμπληρώστε:
  • Network Name: QBFT_Besu_EduNet
  • RPC URL: https://rpc.dimikog.org/rpc/
  • Chain ID: 424242
  • Currency Symbol: EDU-D
  • Block Explorer: https://explorer.dimikog.org

Πατήστε Save και επιβεβαιώστε ότι το MetaMask εμφανίζει το δίκτυο “Besu EduNet”.

💧 3. Απόκτηση EDU-D Tokens (Faucet)

  1. Ανοίξτε το πορτοφόλι σας στο Besu EduNet.
  2. Επισκεφθείτε το faucet:
    • https://faucet.dimikog.org
  3. Επικολλήστε τη διεύθυνση του πορτοφολιού σας (0x...).
  4. Πατήστε Send EDU και περιμένετε μερικά δευτερόλεπτα.
  5. Ανοίξτε το MetaMask για να δείτε το νέο σας υπόλοιπο.

Αυτά τα tokens θα χρειαστούν για να πληρώσετε τα gas fees κατά το mint του NFT.

🎓 4. Mint του Course Starter NFT

  1. Μεταβείτε στη σελίδα mint: https://dimikog.github.io/besu-access-nft/
  2. Contract Address: 0x5Ad2E39FAeE577D233059902eE94bc8df66d69c9
  3. Πατήστε Connect Wallet → επιλέξτε MetaMask.
  4. Πατήστε Mint NFT και περιμένετε επιβεβαίωση συναλλαγής.
  5. Ελέγξτε το NFT σας στο Blockscout ή μέσα από το MetaMask.

Αυτό το NFT λειτουργεί ως ψηφιακή απόδειξη συμμετοχής στο μάθημα Blockchain & DLT Technologies από το M.Sc Cybersecurity.
